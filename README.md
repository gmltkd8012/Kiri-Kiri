# Kiri-Kiri
모임 약속 날짜 투표 및 공유 앱

## 기술 스택
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Firebase (Firestore)

## Getting Started

```bash
npm install
npm run dev
```

[http://localhost:3000](http://localhost:3000)에서 확인할 수 있습니다.

## 주요 기능
- 투표방 생성 (초대코드 방식)
- 캘린더 기반 날짜 투표 (복수 선택)
- 투표 결과 확인 및 날짜 확정
- 확정 날짜 메모 기능
- 카카오톡 공유
